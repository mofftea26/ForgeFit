# Features

- **Programs List** (Home): create, edit, delete, empty state, FAB.
- **Create Program**: stepper flow with cover image & review.
- **Program Details**: phases switcher, days list, menu (edit/export/share/delete).
- **Program Editor**: phase bar, add/remove days, rich WorkoutDay editor.
- **Workout Day Editor**: banner image, title/duration, description, targets, equipment, series builder.
- **Series Builder**: exercises (A1, A2...), sets (type/reps/rest/tempo), trainer notes.
- **Rest Day Editor**: simple description & note.
- **Day View**: read-only version with chips & sets.
